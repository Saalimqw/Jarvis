# JARVIS Voice Assistant
A fast, interruptible voice assistant powered by Groq's Qwen 3 32B model. Say "Jarvis" to activate, issue commands like "open YouTube" or "launch calculator", and interrupt ongoing speech with "Jarvis stop".

JARVIS Demo (Features)-

Wake Word Detection: Activates on "Jarvis"

Interruptible Speech: Say "Jarvis stop" anytime

Groq AI Integration: Qwen 3 32B for smart command processing

Tool Calling: Opens websites, launches programs

Real-time Listening: Auto-adjusts to ambient noise

Threaded TTS: Non-blocking speech synthesis
